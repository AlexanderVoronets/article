Введение
***
Результат выполнения 1 - ого задания содержит два php скрипта. Первый файл index.php содержит основной код выполнения задания, а второй файл - article.php содержит вывод полного текста статьи.
***
Файл index.php содержит переменную $articleText в которой хранится полная версия статьи.  Далее строку обрезаем до 200 символов. После с помощью rtrim удаляем символы "!,.-" которые могли оказатся после обрезки строки в конце. Обработанную строку разбиваем на отдельные слова с помощью ф-ции explode и сохраняем его в $massArticle. С помощью ф-ции count обращаемся к последним 3 м элементам массива $massArticle и формируем ссылку. Затем запоминаем количество символов в ссылке. После в $articleLink кладем фрагмент статьи без 3 х последних слов. В переменной $articleLinkGET формируем текст запроса для перехода на страницу article.php. И выводим сформированную строку, содержащую короткий фрагмент статьи и ссылку на неё полную версию.
***
В файле article.php мы обращаемся к массиву $_GET и выводим полный текст статьи.
***
Проблема, с которыми может столкутся наш php скрипт заключается в том что мы передаем полный текст статьи  с помощью метода GET который имеет ограничение по длинне строки запроса на web сервере с которого запускаеся данный скрипт. При передаче методом GET очень длинной статьи, можно получить ошибку типа: "the requested url's length exceeds the capacity limit for this server").
